# TRPG Dice App — Astaroth's Bones

3D TRPG ダイスロール SPA (React + Three.js + Tone.js)。

## セットアップ

```bash
npm install
```

## 開発

```bash
npm run dev
```

`http://localhost:5173/` が自動で開きます。

## テスト・ビルド

```bash
npm test
npm run build
```
